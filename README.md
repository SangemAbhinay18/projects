# Simon Says Game

This is a simple implementation of the classic Simon Says game built using HTML, CSS, and JavaScript. The game features four colored boxes, each of which lights up in a random sequence. The player must replicate the sequence in the same order to continue playing. If the player makes a mistake, the game ends.

## Features
- Four colored boxes that light up in a random sequence.
- The player must repeat the sequence of colors in the exact order.
- The game will progressively add a new color to the sequence after each successful round.
- Game over is triggered when the player makes a mistake.

## How to Play
1. Start the game by clicking the "Any" button.
2. Watch the sequence of colors light up one at a time.
3. Repeat the sequence by clicking the colored boxes in the same order.
4. Each time you repeat the sequence correctly, a new color is added to the sequence.
5. The game ends when you make a mistake or fail to follow the correct sequence.
6. Display the Score.
